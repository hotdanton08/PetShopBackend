## 簡介

本產品為練習用 Side Project 

主要分成前台和後台

前台頁面有首頁(未登入)、登入畫面、註冊頁面1、註冊頁面2、首頁已登入、用戶中心、產品頁面、購物車、結帳、信用卡資訊、客戶服務

後台頁面有產品管理、訂單管理、用戶管理、內容管理

## 安裝

提供安裝和設置專案的步驟。

1. 克隆專案到本地：

    ```bash
    git clone https://github.com/hotdanton08/myshop-backend2.git
    ```

2. 進入專案目錄：

    ```bash
    cd myshop-backend2
    ```

3. 安裝依賴：

    ```bash
    npm install
    ```
